---------------------------------
MazeGame Readme
---------------------------------

MazeGame randomly generates a maze and allows the player to navigate through the maze, turn on/off features like lighting, day/night, flash light, fog effect, zoom in and out and collisions

Movement
Forward			- W, left stick up
Backward		- S, left stick down
Left			- A, left stick left
Right			- D, left stick right

Toggle Fog 		- Q, X button 
Toggle Day/Night	- E, Right Shoulder button
Toggle Flash		- F, Left Shoulder button
Toggle Collision	- C, Y button
Toggle Wall		- R, DPad Down button
Zoom in			- Z, B button
Zoom out		- Shift Z, A button

Reset to start position - Home, Start button
Quit game		- Escape, Back button
